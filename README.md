# GPSH
This is a mobile green energy container house that integrates solar power generation, energy storage, and smart control, and features deployable wings to expand its space and light-absorbing area.

## 最簡單的照片／影片更新方式

1. 在 GitHub 開啟 `public/media`，進入要更新的分類。
2. 按 **Add file → Upload files**，把檔案直接拖進瀏覽器並提交。
3. GitHub Actions 會自動更新媒體清單，Vercel 隨後自動部署。

分類資料夾：

- `exterior`：外觀照片
- `interior`：室內照片
- `floor2`：二樓照片
- `videos`：影片（建議 MP4）

照片支援 JPG、PNG、WebP、GIF、AVIF；影片支援 MP4、WebM、MOV、M4V。檔名可直接使用中文，但請避免重複檔名。
